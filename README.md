# Vehicle List

## Local Setup

From the Command Line:

```sh
$ git clone git@github.com:calebpollman/vehicle-list.git
$ cd vehicle-list
$ cd yarn install
```

## Run Locally

From the Command Line:

```sh
$ yarn start
```

## Run Tests

From the Command Line:

```sh
$ yarn test
```
